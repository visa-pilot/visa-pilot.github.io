#### Data Scope

This GitHub Pages site is a product information page for Visa Pilot Extension and does not provide account registration or direct data submission.

#### Extension Data Behavior (Product Overview)

Visa Pilot is designed as local-first for persisted user data:

1. Profile, trip, memory, and settings are stored locally in browser storage.
2. Filling actions are user-confirmed (human-in-the-loop).
3. The extension does not auto-submit external official forms.

#### Network Inference Notice

AI responses may require network inference depending on provider mode.
In backend/BFF mode, requests are proxied through the configured service.
In direct provider mode, requests are sent to the configured API provider.

#### Policy Changes

This page may be updated as product behavior evolves. For source-level details, review the repository documentation.
